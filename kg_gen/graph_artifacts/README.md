# Graph Artifacts

This directory keeps the public-safe graph outputs used to compare the two graph construction variants in the research project.

It intentionally contains only compact graph-level artifacts:

- `entities.parquet`
- `relationships.parquet`
- `stats.json`

The full GraphRAG workspaces, source-derived input texts, cache files, LanceDB indexes, logs, embeddings, pickle files, and generated community reports are intentionally omitted from the public snapshot.

## Variants

### `microsoft_graphrag/`

Baseline graph output generated by applying the Microsoft GraphRAG pipeline to the agricultural document corpus.

Retained files:

- `entities.parquet`
- `relationships.parquet`
- `stats.json`

### `clustered_agri_kg/`

Final graph output from the SNU-KG / Clustered-Agri-KG pipeline after integrating kg-gen-style graph extraction and semantic entity/relation consolidation.

Retained files:

- `entities.parquet`
- `relationships.parquet`
- `stats.json`

These files are equivalent to the final normalized graph artifacts also retained under `kg_gen/snu_kg_output/final/`.

## Interpretation

Use this directory when comparing the graph construction outputs behind:

- Microsoft GraphRAG baseline retrieval
- Clustered-Agri-KG GraphRAG retrieval

The files are provided as research artifacts, not as a complete rehydration of the original experiment workspace.
