## 1. Multi-hop Reasoning (다중 홉 추론)

### 1.1 개요
- **정의**: 2개 이상의 엔티티를 거쳐가며 연결되는 관계를 추론하는 질문
- **추출된 패턴**: 총 100개 (2-hop: 82개, 3-hop: 18개)
- **난이도**: Easy (2-hop) ~ Medium (3-hop)

### 1.2 패턴 특징

#### 2-hop 패턴 (82개)
- **구조**: A → B → C
- **평균 품질 점수**: 6.9/8
- **주요 템플릿**:
  1. 작물 → 환경 요인 → 환경 요인 (12개)
  2. 환경 요인 → 환경 요인 → 환경 요인 (7개)
  3. 작물 → 환경 요인 → 기타 (다양한 조합)

**대표 예시**:
```
경로: 토마토 → LOW TEMPERATURE → 생육
타입: 작물 → 환경 요인 → 환경 요인
관계1: "토마토와 온도는 밀접한 관계가 있으며..."
관계2: "온도는 작물의 생육에 중요한 환경 요인으로 작용합니다."
```

#### 3-hop 패턴 (18개)
- **구조**: A → B → C → D
- **평균 타입 다양성**: 2.5개 고유 타입
- **복잡도**: 더 길고 다양한 경로를 통한 추론 필요

### 1.3 주요 브릿지 노드 분석
**브릿지 노드**: 2-hop 경로에서 중간 연결 역할

| 노드 | 타입 | 브릿지 역할 횟수 | 연결 다양성 |
|------|------|------------------|-------------|
| 저온 | 환경 요인 | 7회 | 2개 소스 타입 × 1개 타겟 타입 |
| HIGH TEMPERATURE | 환경 요인 | 7회 | 2개 소스 타입 × 3개 타겟 타입 |
| 온난화 | 환경 요인 | 4회 | 2개 소스 타입 × 3개 타겟 타입 |
| 생육 | 환경 요인 | 4회 | 1개 소스 타입 × 2개 타겟 타입 |

### 1.4 의미론적 패턴 분석
- **인과관계**: 18개 (60%) - "때문", "영향" 등의 키워드 포함
- **속성관계**: 1개 (3%) - "특징", "속성" 관련
- **연관관계**: 11개 (37%) - 일반적 연결 관계

### 1.5 생성할 QA 예시

**2-hop 질문 템플릿**:
1. "{A}와(과) {C}의 관계는 무엇인가? ({B}를 통해)"
2. "{A}이(가) {C}에 어떤 영향을 미치는가?"
3. "{A}에서 {B}를 거쳐 {C}에 이르는 연결고리는?"

**예시 질문-답변 쌍**:
```
Q: 토마토 재배에서 저온이 생육에 미치는 영향 경로는 무엇인가요?
A: 토마토는 온도에 민감한 작물로, 특히 저온(LOW TEMPERATURE)은 토마토의 생장과 초기 활착에 
   중요한 영향을 미칩니다. 저온은 작물의 생육에 부정적인 영향을 주는 환경 요인으로 작용하여, 
   결과적으로 토마토의 전체적인 생육 상태에 영향을 미치게 됩니다.
```

## 2. Community Synthesis (커뮤니티 기반 합성)

### 2.1 개요
- **정의**: 같은 커뮤니티에 속한 엔티티들의 관계와 커뮤니티 전체의 주제를 통합적으로 이해하는 질문
- **추출된 패턴**: 총 38개
- **난이도**: Medium ~ Hard (커뮤니티 레벨에 따라)

### 2.2 커뮤니티 구조 분석

#### 레벨별 특성
| 레벨 | 커뮤니티 수 | 평균 크기 | 평균 타입 다양성 | 평균 내부 밀도 |
|------|-------------|-----------|------------------|----------------|
| 0 | 14개 | 16.0 엔티티 | 4.5개 타입 | 0.48 |
| 1 | 24개 | 15.1 엔티티 | 5.0개 타입 | 0.47 |

### 2.3 주요 커뮤니티 테마
1. **고추 생육 온도와 묘목 성장**
2. **농업 기술 지원 시스템과 분석 시설**
3. **고추 및 파프리카 연구 커뮤니티**
4. **농촌진흥청과 기후 변화 연구 네트워크**
5. **충남농업기술원 과채연구소와 영농기술 정보**

### 2.4 내부 관계 패턴
- **협력/연계**: 15개 (22%) - "함께", "공동" 등
- **계층/분류**: 2개 (3%) - "속한", "포함" 등
- **인과/영향**: 51개 (75%) - "영향", "때문" 등

### 2.5 대표 커뮤니티 분석

**예시: 전주시 농업과 질병 관리 커뮤니티**
- **규모**: 30개 엔티티
- **타입 다양성**: 6개 타입
- **주요 구성원**: 농촌진흥청 국립원예특작과학원, 토마토 뿔나방, 연구자들
- **요약**: "전주시는 농촌진흥청이 위치한 도시로, 농업 관련 연구와 지원을 통해..."

### 2.6 생성할 QA 예시

**질문 템플릿**:
1. "{커뮤니티명}에 속한 주요 요소들의 상호작용은?"
2. "{커뮤니티명}의 핵심 주제와 구성원들의 역할은?"
3. "{커뮤니티명}에서 {엔티티A}와 {엔티티B}의 관계는?"

**예시 질문-답변 쌍**:
```
Q: 농촌진흥청과 기후 변화 연구 네트워크의 주요 연구 주제와 참여 기관들의 역할은 무엇인가요?
A: 이 네트워크는 기후 변화가 농업에 미치는 영향을 연구하는 커뮤니티입니다. 농촌진흥청을 중심으로 
   FAO, 국제 기후학 저널 등이 참여하여 가뭄과 염분 스트레스, 지중해형 생태계의 변화 등을 연구합니다. 
   특히 기후 변화가 작물 품질에 미치는 직접적인 영향을 분석하고 지속 가능한 농업 생산 방안을 모색합니다.
```

## 3. Cross-context Integration (교차 컨텍스트 통합)

### 3.1 개요
- **정의**: 여러 text unit에 걸쳐 분산된 정보를 통합해야 답할 수 있는 질문
- **추출된 패턴**: 총 100개
- **난이도**: Medium ~ Hard (통합해야 할 text unit 수에 따라)

### 3.2 패턴 구조

#### Text Unit 분포
- **평균 관련 text units**: 4.5개
- **최대 관련 text units**: 7개
- **가장 활발한 text unit**: 상위 10개가 전체 패턴의 40% 차지

#### 앵커 엔티티 분석
앵커 엔티티: Cross-context 패턴의 중심이 되는 엔티티

| 앵커 타입 | 패턴 수 | 평균 관련 엔티티 | 평균 text unit |
|-----------|---------|-------------------|----------------|
| 작물 | 25개 | 3.2개 | 4.8개 |
| 환경 요인 | 20개 | 2.8개 | 4.2개 |
| 농업 기술 | 15개 | 3.5개 | 5.1개 |

### 3.3 통합 패턴 유형
1. **근거리 통합** (30%): 같은 문서 내 가까운 units
2. **원거리 통합** (25%): 같은 문서 내 먼 units
3. **교차 문서 통합** (20%): 다른 문서의 units
4. **다중 컨텍스트** (25%): 5개 이상의 다양한 units

### 3.4 생성할 QA 예시

**질문 템플릿**:
1. "{앵커 엔티티}와 관련된 정보를 여러 컨텍스트에서 종합하면?"
2. "{앵커 엔티티}에 대한 다양한 관점의 정보를 통합하여 설명하시오"
3. "여러 문서에서 언급된 {앵커 엔티티}의 특성을 종합하면?"

**예시 질문-답변 쌍**:
```
Q: 키토산의 농업적 활용에 대해 여러 연구 결과를 종합하여 설명해주세요.
A: 키토산은 다양한 농업 분야에서 활용되고 있습니다. 첫째, 토마토 재배에서는 잎곰팡이병 방제에 
   효과적이며, 특히 SH-1 제형은 강력한 질병 제어 활성을 보입니다. 둘째, 딸기 재배에서는 
   잿빛곰팡이병과 역병균 생장을 억제하여 병 저항성을 향상시킵니다. 셋째, 고추와 파프리카에서는 
   품질 향상과 저장성 개선 효과가 있습니다. 이처럼 키토산은 여러 작물에서 병해 방제와 품질 향상을 
   동시에 달성할 수 있는 친환경 농업 자재로 평가받고 있습니다.
```

## 4. Implicit Relationships (암시적 관계)

### 4.1 개요
- **정의**: 그래프에서 직접 연결되지 않았지만 커뮤니티나 공통 이웃을 통해 관련성이 있는 엔티티들의 관계
- **추출된 패턴**: 총 69개
- **난이도**: Medium ~ Hard

### 4.2 패턴 유형별 분포

| 패턴 유형 | 개수 | 비율 | 특징 |
|-----------|------|------|------|
| 커뮤니티 기반 | 60개 | 87% | 같은 커뮤니티 내 직접 연결되지 않은 노드 |
| 공통 이웃 기반 | 8개 | 12% | 2개 이상의 공통 이웃을 가진 노드 쌍 |
| 복합적 관계 | 1개 | 1% | 여러 커뮤니티에서 동시에 나타나는 관계 |

### 4.3 커뮤니티 기반 암시적 관계
- **주요 커뮤니티**: Level 0에서만 추출 (더 구체적인 관계)
- **평균 공통 이웃 수**: 1.8개
- **주요 엔티티 타입 조합**:
  1. 빈 타입("") ↔ 연구 연구 (9회)
  2. 빈 타입("") ↔ 조직 (6회)
  3. 빈 타입("") ↔ 환경 요인 (6회)

### 4.4 공통 이웃 기반 암시적 관계
- **공통 이웃 규모**:
  - 소규모 (2개): 3개 패턴
  - 중규모 (3-5개): 4개 패턴
  - 대규모 (6개+): 1개 패턴
- **공통 이웃의 주요 타입**: 환경 요인, 작물, 농업 기술

### 4.5 암시적 관계 강도 평가
- **고강도 관계**: 69개 중 45개 (65%)
- **평균 강도 점수**: 5.2/8
- **강도 결정 요인**: 공통 이웃 수, 커뮤니티 맥락, 엔티티 타입 일치도

### 4.6 생성할 QA 예시

**질문 템플릿**:
1. "{엔티티A}과 {엔티티B}의 {커뮤니티명} 내에서의 연관성은?"
2. "{엔티티A}과 {엔티티B}가 공통으로 관련된 {N}개 요소를 통한 간접적 관계는?"
3. "{엔티티A}과 {엔티티B}가 {N}개 커뮤니티에서 보이는 다면적 연관성은?"

**예시 질문-답변 쌍**:
```
Q: 농촌진흥청과 FAO가 기후 변화 연구 네트워크 내에서 어떤 간접적 연관성을 가지고 있나요?
A: 두 기관은 직접적으로 연결되어 있지 않지만, 같은 커뮤니티 내에서 중요한 역할을 합니다. 
   농촌진흥청은 한국의 농업 연구를 주도하며, FAO는 국제적 차원에서 기후 변화와 농업의 관계를 
   연구합니다. 이들은 'CLIMATE CHANGE', 'GLOBAL ENVIRONMENTAL CHANGE' 등의 연구 주제를 
   공통으로 다루며, 가뭄과 염분 스트레스가 농업에 미치는 영향을 연구하는 점에서 간접적으로 
   연결됩니다. 특히 지속 가능한 농업 생산이라는 공통 목표를 추구합니다.
```

## 5. Causal Chains (인과 관계 체인)

### 5.1 개요
- **정의**: 원인과 결과의 연쇄적 관계를 추적하는 질문
- **추출된 패턴**: 총 100개
- **난이도**: Easy (직접) ~ Hard (다단계/복합)

### 5.2 패턴 유형별 분포

| 패턴 유형 | 개수 | 비율 | 특징 |
|-----------|------|------|------|
| 직접적 인과관계 | 50개 | 50% | A → B (1-hop) |
| 인과관계 체인 | 40개 | 40% | A → B → C (2-3 hop) |
| 복합 인과관계 | 10개 | 10% | 분기형/수렴형 |

### 5.3 직접적 인과관계 분석

#### 인과관계 타입 분포
- **cause (원인)**: 18개 - "때문", "인해", "원인" 등
- **effect (결과)**: 15개 - "결과", "영향", "변화" 등
- **mechanism (메커니즘)**: 12개 - "통해", "으로써" 등
- **general (일반)**: 5개

#### 가중치 분석
- **평균 가중치**: 28.5
- **중앙값**: 22.0
- **최대 가중치**: 136 (키토산 → SH-1)

#### 주요 인과관계 타입 변환
1. 환경 요인 → 작물 (15회)
2. 농업 기술 → 질병 (10회)
3. 환경 요인 → 환경 요인 (8회)

### 5.4 인과관계 체인 분석

#### 체인 길이별 분포
- **2-hop 체인**: 25개 (62.5%)
- **3-hop 체인**: 15개 (37.5%)

#### 중간 매개 노드 분석 (2-hop)
- **주요 중간 노드 타입**: 환경 요인 (60%), 농업 기술 (25%)
- **브릿지 역할**: 원인을 결과로 전달하는 매개체

### 5.5 복합 인과관계 패턴

#### 분기형 (Branching) - 5개
- **구조**: 하나의 원인 → 여러 결과
- **평균 결과 수**: 3.2개
- **예시**: 고온 → {생육 저하, 낙화, 품질 저하}

#### 수렴형 (Converging) - 5개
- **구조**: 여러 원인 → 하나의 결과
- **평균 원인 수**: 2.8개
- **예시**: {온도, 습도, 통풍} → 역병 발생

### 5.6 생성할 QA 예시

**질문 템플릿**:

1. **직접 인과**: "{원인}이(가) {결과}에 미치는 인과적 영향은?"
2. **인과 체인**: "{시작점}에서 시작된 연쇄적 영향이 {종점}에 이르는 경로는?"
3. **분기형**: "{원인}이(가) 야기하는 다양한 영향들은?"
4. **수렴형**: "{결과}에 영향을 미치는 다양한 요인들은?"

**예시 질문-답변 쌍**:

```
Q: 고온이 토마토 재배에 미치는 다단계 영향 경로를 설명해주세요.
A: 고온은 토마토 재배에 연쇄적인 영향을 미칩니다. 
   1단계: 고온 → 생육 불균형 (고온으로 인해 영양생장과 생식생장의 균형이 깨짐)
   2단계: 생육 불균형 → 낙화 증가 (불균형한 생육으로 꽃이 떨어지는 현상 증가)
   3단계: 낙화 증가 → 수확량 감소 (착과수 감소로 최종 수확량이 줄어듦)
   이러한 연쇄 반응을 방지하기 위해 고온기 재배 시 착과수 조절, 측지 방임 등의 
   재배 기술이 필요합니다.
```

### 6.3 GraphRAG의 차별점

#### 전통적 RAG 대비 장점
1. **Multi-hop**: 문서 간 연결된 정보 추적 가능
2. **Community**: 주제별 클러스터링된 정보 통합
3. **Cross-context**: 분산된 정보의 체계적 통합
4. **Implicit**: 명시되지 않은 숨은 관계 발견
5. **Causal**: 복잡한 인과관계 체인 추적

### 6.4 다음 단계 (Phase 4-6)

1. **Phase 4: 질문 생성**
   - 각 패턴별 템플릿 기반 자연어 질문 생성
   - LLM을 활용한 질문 다양화 및 품질 검증

2. **Phase 5: Ground Truth 생성**
   - 패턴의 source_texts와 관계 정보 기반 정답 생성
   - 전문가 검증을 위한 근거 포함

3. **Phase 6: 최종 데이터셋 구성**
   - QA 쌍 통합 및 중복 제거
   - 난이도별, 유형별 균형 맞추기
   - 최종 JSON 형식 출력
